# 👥 HR Dashboard

## Overview

This project is an interactive HR Analytics dashboard developed in Power BI to help organizations analyze employee data and support better workforce decisions.

---

## Business Problem

Human Resource departments need quick insights into workforce demographics, employee distribution, salaries, and other HR metrics to improve planning and decision-making.

---

## Project Objective

This dashboard was created to:

- Analyze employee demographics.
- Monitor salary distribution.
- Evaluate employee positions and departments.
- Support HR decision-making using interactive visualizations.

---

## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel
