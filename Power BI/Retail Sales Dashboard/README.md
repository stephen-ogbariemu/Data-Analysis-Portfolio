# Retail Sales Dashboard

## Overview

This project is an interactive Power BI dashboard developed to analyze retail sales performance. It provides insights into sales trends, customer demographics, product categories, and purchasing behavior to support data-driven business decisions.

## Business Objective

The goal of this dashboard is to:
- Monitor overall sales performance.
- Analyze revenue trends over time.
- Compare sales across product categories.
- Understand customer demographics.
- Support strategic business decisions.

## Tools Used

- Power BI
- Microsoft Excel
- Power Query
- DAX

## Dataset

The dataset contains retail sales transaction records, including customer information, product categories, sales amounts, transaction dates, and quantities sold.
