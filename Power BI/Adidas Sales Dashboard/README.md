# Adidas Sales Dashboard

## Overview

This project is an interactive Power BI dashboard developed to analyze Adidas sales performance across different regions, products, and sales channels.

---

## Business Problem

Businesses need to monitor sales performance, identify high-performing products, and understand regional trends to improve profitability and support strategic decision-making.

---

## Project Objectives

- Analyze total sales performance.
- Compare sales across regions.
- Identify top-performing products.
- Monitor retailer performance.
- Track sales trends.

---

## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel
