# Apple Sales Dashboard

## Overview

This project is an interactive Microsoft Excel dashboard developed to analyze Apple product sales performance. It provides insights into sales trends, product performance, and business metrics through charts, PivotTables, and slicers.

---

## Business Problem

Businesses need to monitor sales performance, identify top-performing products, and analyze trends to support informed business decisions.

---

## Project Objectives

- Analyze total sales performance.
- Compare product sales.
- Monitor sales trends.
- Evaluate key performance indicators.
- Support data-driven decision-making.

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
