# 🥤 Coca-Cola Sales Performance Dashboard

## 📌 Project Overview

This project is an interactive Microsoft Excel dashboard developed to analyze Coca-Cola sales performance across beverage brands, retailers, and reporting periods. The dashboard transforms raw sales data into meaningful business insights using PivotTables, PivotCharts, slicers, and dynamic Excel features.

---

## 🎯 Business Problem

Sales managers require a centralized dashboard to monitor sales performance, compare year-over-year results, identify top-performing brands and retailers, and evaluate operating margins. Without a visual reporting tool, identifying trends and making informed decisions becomes time-consuming.

---

## 🎯 Project Objectives

This dashboard was developed to:

- Compare sales performance between **2022 and 2023**.
- Measure year-over-year sales growth using variance analysis.
- Evaluate the performance of different Coca-Cola beverage brands.
- Identify the highest-performing retail partners.
- Analyze quarterly sales trends and operating margins.
- Present business insights through an interactive dashboard.

---

## 📊 Dashboard Highlights

The dashboard includes:

- Year-over-Year Sales Comparison (2022 vs 2023)
- Sales by Beverage Brand
- Sales by Retailer
- Quarterly Sales Analysis
- Operating Margin by Quarter
- Interactive Filters (Slicers)
- KPI Summary and Variance Analysis
